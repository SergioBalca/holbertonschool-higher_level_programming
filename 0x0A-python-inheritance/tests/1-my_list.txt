The ``1-my_list`` module
============================

Using ``print_sorted``
---------------------

First import ``print_sorted`` from the ``1-my_list`` module:

	>>> MyList = __import__('1-my_list').MyList

Now use it:

	>>> my_list= MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

	>>> my_list = MyList()
	>>> my_list.append(-1)
	>>> my_list.append(0)
	>>> my_list.append(10)
	>>> my_list.append(2)
	>>> print(my_list)
	[-1, 0, 10, 2]
	>>> my_list.print_sorted()
	[-1, 0, 2, 10]

	>>> my_list = MyList()
	>>> my_list.append(-3)
	>>> my_list.append(-5)
	>>> my_list.append(-1)
	>>> my_list.append(-2)
	>>> print(my_list)
	[-3, -5, -1, -2]
	>>> my_list.print_sorted()
	[-5, -3, -2, -1]
